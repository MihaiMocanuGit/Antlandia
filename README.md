# Description

This app's purpose is to simulate the behaviour of an ant colony. 





# Compiling the app
## Needed Dependencies: 
* SFML 2
* c++17
* make
* g++
## Compile and Run
Note: This has only been tested on Linux, I have not tried to run it on Windows yet.

Use `make` to compile and `./antSim` to run.

Shortcuts: 
* `make run` to build and run
* `make clean` to delete all generated objects and executable 

# Features
* (to be implemented) Multi-threaded
* The world is composed of Chunks so as to speed up simulation speed when dealing with large numbers of objects
